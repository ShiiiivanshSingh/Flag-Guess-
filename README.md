<h2>✨ Features</h2>

Here're some of the project's best features:

*   Difficulty Selector
*   Gameplay Interface
*   Game Over Screen
*   Responsive UI
*   Flag Image displayed for guessing.
*   Lives (hearts) track remaining attempts
*   Guess Input box for country guesses.
*   Hint area for assistance.
